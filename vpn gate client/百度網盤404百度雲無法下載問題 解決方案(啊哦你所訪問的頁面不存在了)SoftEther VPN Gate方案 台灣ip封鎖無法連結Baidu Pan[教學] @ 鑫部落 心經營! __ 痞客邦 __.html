      [教學]百度網盤404百度雲無法下載問題 解決方案(啊哦你所訪問的頁面不存在了)SoftEther VPN Gate方案 台灣ip封鎖無法連結Baidu Pan[教學]

       

          2018/3 月初左右，台灣百度網盤的用戶都會遇到一個狀況，從論壇(網站)取得分享的度盤連結，輸入後卻看不到下載點，只能看得到封面圖的 404 錯誤，或稱你所訪問的頁面不存在了 (你所访问的页面不存在了)。

       

       

      論壇說法是暫時無法使用。

       

      有傳聞說這是政治問題，但無法證實。

       

       

      以往是一路暢通的，現在不通原因是台灣 ip 被檔，所以過不了第一關，只會回應你 404 (前段 X)。然而，如果自己網盤的檔案要下載，仍是沒有問題的 (後段 OK)。

       

      問題之狀況與解法，分析如下圖。

       

         

       

      解法為透過 VPN (虛擬私人網路)，讓雲盤網站當成來訪者是來自大陸 ip 的電腦。

       

      安裝 SoftEther VPN Client + VPN Gate Client 插件 後，於主畫面中優先選一台位於大陸的 VPN 中繼服務器。

       

       

      此 app 很便利，連線完成就全部走 VPN 路徑，斷開就立即回到家裡網路。

       

      中繼服務器有快有慢，連接後才知道。若反應太慢，請趕快換另外一個。

       

       

      SoftEther VPN 之詳細安裝與使用教學，請參這一篇: 免費 SoftEther VPN 翻牆軟體 日本筑波大學 VPN Gate 解決封鎖 403 Forbidden 404

       

      試看看。

       

      OK 了!

       

       

      成功透過 VPN 穿越空間到對岸，俗稱翻牆。

       

      VPN 中繼服務器不一定只能走 China，還可以繞道其他沒被阻擋的國家，實際試過美國、日本、韓國、泰國、香港、俄國各地，皆可以過。另外它的協定有 TCP 或 UDP 可選，需要配合 VPN 中繼服務器之設定。可以先用 TCP 測，不通再換 UDP。若還是不通，就換別台 VPN 中繼服務器，不必浪費時間。

       

      以前到廣州市出差，有些東西也必須透過 VPN 才能到台灣。現在情況變了，還得翻牆才能過去。

       

       

      檔案確實可以保存到自己網盤內，這一點是重要突破。因為從自己雲盤傳輸到家裡硬碟這段路徑，原本就沒有被阻擋，可以存就等於可以抓。

       

       

      請注意下面 1.13G 的檔案，這是剛剛現抓的。

       

       

       

      要抓檔案，請記得先斷開 VPN 路徑，切回平常家裡網路，否則將會很慢。

       

      透過雲盤 PC 版下載到自己硬碟，速度還不錯。這段原本就能通過，現在還是暢行無阻。

       

       

      極速下載也沒問題。

       

      因為透過 HiNet Proxy，速度可以到 4-6 Mb/s。(請參考這一篇百度網盤 Proxy 設定，嘗試從你的 ISP 就近取得快取副本，能找到的話，傳輸不再姍姍來遲，速度可以到達 4-6 Mb/s) 

       

      萬一遇到很難下載的超長時慢速檔，請參考這篇教學: 百度網盤低速下載問題 突破超長時抓擋緩慢之解決方案(低傳輸率)[教學]。

        

        

       

      根據讀者 nancy0218 在本文問與答的分享，百度客服似乎還沒有任何解除封鎖台灣 ip 的時間表。目前想從台灣直接看到網盤分享內容，即不跳出 404 畫面，唯有「好友分享」一途。問題是，「好友分享」是指彼此有互加好友的百度網盤用戶們，但我們一般都是抓公開分享(不限定好友)的資源。這塊其實才是最主要的部份，所以「好友分享」無法帶來很大突破或幫助，實況上還是等同於封鎖的。看起來，VPN 仍是最可行與便利的解答。

       

      將以上的說明，濃縮成一張圖，就是底下這樣:

       

        

       

      VPN 有時比較慢，所以載入頁面會等很久。但它畢竟是好心網友提供的免費資源，請耐心等待一下。另外，建議各位讀者把抓下載點的工作集中，再一批用 VPN 搬到雲盤，才不會頻繁地切換網路。

       

      何時全面恢復不知道，但目前確定可用此方案應急。

        

      (註: 2016 百度官方已將一般個人用戶用的"百度雲"名稱更新為"百度網盤"，"百度雲"名稱則轉給商務用戶)

        (1)
        百度的檔案要分享一定不能是台灣的 IP
        只要是台灣的 IP 的分享連結全部會被封鎖

        (2)
        要分享檔案一定要先翻牆成大陸的 IP
        這樣取得的分享連結才不會被封鎖

        (3)
        而且分享的檔案連結不能是加密分享
        因為加密分享的提取密碼也會被百度封鎖

        總結
        台灣的 IP 想要分享檔案
        一定要先翻牆成大陸的 IP
        而且檔案只能設定成公開分享
