[下載] VPN Gate Client 免費不限流量 VPN 連線工具（安裝、使用教學）

一般生活在台灣的朋友大概比較少用到 VPN 服務，因為台灣連線環境沒有什麼限制，要連接到全世界的網站幾乎是暢行無阻，不過有時遇到一些國家限定的網路服務，就必須利用 VPN 來作跳板，或者是想玩某些遊戲、下載區域 LINE 貼圖等等，其實都可以利用 VPN Gate Client 來輕鬆實現。

VPN Gate 是日本國立筑波大學的一項學術研究、實驗計畫，主要目的為「推廣全球分散式公共 VPN 中繼伺服器知識」，簡單來說，VPN Gate 網站上會列出目前可用的公共 VPN 中繼伺服器列表，使用者能自由選擇要進行連線的伺服器，無須註冊、也不用付費，這些 VPN 都是由來自世界各地的自願者提供，會不斷更新。

下載帶有 VPN Gate Client 外掛的 SoftEther VPN Client 工具，可以讓 Windows 輕鬆連接到 VPN Gate 上的中繼伺服器，這工具是由日本筑波大學開發的免費軟體，目前也有提供中文介面，如果你對於 VPN 的用量不大，只有偶爾需要用一下，那麼 VPN Gate 就是一個不錯的選擇，速度快、穩定性夠，更新頻繁，而且沒有流量限制。

VPN Gate Client 安裝說明

開啟 VPN Gate 後，點選上方的「下載」，將 SoftEther VPN Client + VPN Gate Client 下載後解壓縮，點選裡面的 vpngate-client-v4… 應用程式來進行安裝。

開啟 SoftEther VPN 安裝嚮導，會有一個簡單的說明，其實在 Mac、iPhone、iPad、Android 等平台也可以使用 VPN Gate ，不過需要自己手動設定（參考教學），還是推薦安裝 SoftEther VPN 來進行設定，速度快、而且也能取得新的 VPN Gate 伺服器列表。

安裝時請選擇第一個「SoftEther VPN Client」，然後按「下一步」。

安裝前還需要選擇路徑、勾選同意使用條款等等。

最後按下「完成」，直接開啟 SoftEther VPN Client 管理工具。

使用教學

開啟 SoftEther VPN Client 管理器後，點擊「VPN Gate 公共 VPN 中繼伺服器」兩下來開啟列表。

接著就能看到目前的所有公共 VPN 中繼伺服器列表，包含 DDNS 唯一主機名稱、IP 位址、國家地區、運行時間、VPN 會話（連線數？）、線路速度和 Ping 值。

我習慣直接按一下「國家/地區」進行排序，這樣就能快速從現有的公共 VPN 中繼伺服器清單找到我要連線的國家，例如你想連到韓國、日本來下載 LINE 貼圖、或是連到美國來瀏覽一些線上影片、節目，就可以直接選擇你要連線的目標後，按下右下角的「連接到 VPN 伺服器」。

因為在某些國家是禁止使用 VPN Gate 服務的，在連接前會跳出注意事項，沒問題的話按下「同意」。

選擇 VPN 協議時請直接用預設的「使用 TCP 協議（Ethernet over HTTPS VPN）（推薦）」選項。

大約會需要幾秒鐘來建立一個 VPN 虛擬網路連線配置，當畫面消失後，就會顯示你取得的 VPN IP 位址。

要確定有沒有連接到你要的 VPN 伺服器，可以開啟 VPN Gate 首頁（http://www.vpngate.net/cn/），中間就會顯示你目前的 IP 位址，我在連接到日本的 VPN 後，IP 來源也變成了 114.160 開頭的位址。

試著在 Windows 命令提示下輸入「ipconfig /all」指令，如果被發配到一個 10.211 開頭的 IP 位址，代表已經成功建立與 VPN Gate 的連結。

不繼續使用、要結束目前的 VPN 連線時，回到 SoftEther VPN Client 管理器，在上方的 VPN Gate Connection 按下滑鼠右鍵，選擇「斷開」就能結束連線，回到你目前的連線環境。

如果你願意，也可以安裝一個軟體（SoftEther VPN Server）來提供自己的 CPU 資源、頻寬給其他需要 VPN 的人使用，VPN 本身採用加密連線通道，即使連線被監聽、封包擷取也無法獲得你所傳輸的內容，例如帳號密碼，所以是非常安全的，有興趣的話不妨參考一下 VPN Gate 網站上的中文說明。
